## This repository is only for code test for Solum Clinic. 

## The answer for the three questions of the coding testing is stored in three folder, A, B and C.
For question 1, the answer is stored in the folder A. Similar to question 1, the answer for question 2 is stored in B. And the application for question 3 is stored in C.

## Please download the whole repository to execute the programs.
Please navigate to corresponding folder in the terminal and execute the command `java MagicalWavesCalculator` for question 1 and `java CargoCraftFleet` for question 2 to execute the java program. For question 3, please first navigate to the corresponding folder in the terminal and execute the command `npm install` to install the dependencies of the application and then execute `npm start` to execute the application.
